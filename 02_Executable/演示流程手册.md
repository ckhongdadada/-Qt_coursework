# 个人发展规划系统 (PDP) — 演示流程手册

## 启动方式

1. 双击 `start_desktop.bat` 启动桌面客户端
2. 确保 `ai_server.py` 已启动（`start_server.bat`），AI 功能需要此服务

## 一、总览页

**入口**：左侧导航栏 → "总览"

**页面功能**：展示个人发展全景数据，包含8个核心指标、学分进度条、近期建议和学期走势。

**预期数据**：

| 指标 | 预期值 |
|------|--------|
| 已修课程 | 32 |
| 当前 GPA | ~3.52 |
| 目标平均进度 | ~57% |
| 成果数量 | 7 |
| 经历数量 | 8 |
| 角色数量 | 5 |
| 活动数量 | 6 |
| 已修学分 | ~92 |

**演示步骤**：

1. 进入总览页，观察8个指标卡片是否正常显示数值
2. 上方有"当前培养方案"下拉框，切换不同专业（数据科学与大数据技术专业、电子商务专业等），观察学分进度条变化
3. 学分进度条应显示各板块（通修必修、通修选修、学科基础等）的完成比例
4. "近期建议"区域应显示2-3条建议（如"GPA表现优秀"、"成果积累较少"等）
5. "学期走势"区域应显示各学期GPA和平均分
6. 点击"导出总览"按钮 → 选择保存路径 → 确认HTML文件生成
7. 点击"清除数据"按钮 → 确认弹窗 → 数据清空（演示时慎用）

## 二、课程库

**入口**：左侧导航栏 → "课程库"

**页面功能**：管理所有课程数据，支持搜索、筛选、排序、新增、编辑、删除。

**预期数据**：557门课程（537门培养方案 + 20门演示课程）

**演示步骤**：

1. **浏览课程**：表格应显示课程名称、代码、学期、学分、分数、绩点、类别、板块、状态共9列
2. **搜索**：在搜索框输入"演示课程"→ 应只显示20门演示课程
3. **状态筛选**：下拉选择"已完成"→ 显示32门已完成课程（有分数）
4. **类别筛选**：下拉选择"必修"→ 只显示必修课
5. **排序**：切换排序方式为"分数"→ 按分数降序排列
6. **专业筛选**：选择"数据科学与大数据技术专业"→ 只显示该专业课程
7. **新增课程**：点击"新增课程"→ 弹出编辑对话框 → 填写信息 → 保存
8. **编辑课程**：选中一行 → 点击"编辑选中课程"（或双击行）→ 修改分数/状态 → 保存
9. **删除课程**：选中一行 → 点击"删除选中课程"→ 确认删除

**关键验证**：
- 演示课程1-高等数学：92分，4学分，已完成
- 演示课程3-程序设计：95分，4学分，已完成
- 演示课程13-大数据技术：进行中
- 演示课程16-深度学习：未开始

## 九、简历导出

**入口**：左侧导航栏 → "简历导出"

**页面功能**：配置并预览简历，支持导出HTML和复制到剪贴板。

**预期数据**：左侧编辑面板有默认配置，右侧候选面板从数据库自动提取数据。

**演示步骤**：

1. 进入简历页，左侧为编辑面板（可修改姓名、标题、邮箱、电话、求职意向等）
2. 右侧为候选面板，自动从课程、经历、成果、角色中提取数据
3. 点击候选面板中的条目可插入到简历对应分区

5. 点击"刷新预览"→ 重新生成
6. 点击"恢复默认配置"→ 重置为默认值
7. 点击"导出 HTML"→ 选择保存路径 → 生成HTML文件
8. 点击"复制到剪贴板"→ 简历内容复制到剪贴板

## 十三、AI 助手

**入口**：右侧面板 → AI助手

**页面功能**：基于大模型的智能问答，支持上下文选中文本分析。

**演示步骤**：

1. 在任意页面选中文本（如课程名称）
2. 右侧AI面板应自动识别选中文本
3. 在输入框输入问题（如"这个课程对我的发展有什么帮助？"）
4. 点击"发送"→ 等待AI回复
5. AI回复应结合选中文本和数据库上下文

**注意**：AI功能需要 `ai_server.py` 服务运行中。如果服务未启动，系统会使用本地规则模式生成回复。

## 演示建议流程（课堂演示顺序）

1. **总览页**（1分钟）：展示全景数据，切换专业看学分进度
2. **课程库**（1分钟）：搜索"演示课程"，筛选已完成，排序按分数
3. **角色/成就/经历/活动**（各30秒）：快速浏览数据，演示新增/编辑
4. **目标管理**（30秒）：展示进度条，演示更新进度
5. **目标岗位**（30秒）：展示岗位和技能要求
6. **数据导入**（1分钟）：导入演示PDF，展示2秒解析
7. **简历导出**（1分钟）：预览简历，导出HTML
8. **分析报告**（30秒）：展示综合分析
9. **时间轴**（30秒）：展示发展历程
10. **AI助手**（1分钟）：选中文本提问，展示智能回复

总计约8分钟。
