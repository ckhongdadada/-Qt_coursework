# PDP 学业发展规划系统 — 技术说明文档（桌面端版）

---

## 1. 项目概述

### 1.1 项目简介

PDP（Personal Development Planning）学业发展规划系统是一款基于 **Qt 6 + C++17** 的桌面端应用程序，帮助学生系统化管理学业生涯中的课程成绩、目标规划、实践经历、课外活动、成就荣誉、岗位追踪等核心数据，并通过 AI 智能分析提供个性化发展建议。

### 1.2 核心功能

| 序号 | 功能模块 | 说明 |
|------|---------|------|
| 1 | 课程管理 | 课程信息录入、成绩管理、GPA 自动计算（加权/标准两种算法） |
| 2 | 目标规划 | 学期目标设定、里程碑追踪、进度百分比自动计算 |
| 3 | 角色任职 | 学生干部/社团角色管理、任期追踪 |
| 4 | 成就荣誉 | 奖学金、竞赛获奖等成就记录 |
| 5 | 实践经历 | 实习、志愿服务等实践经历管理 |
| 6 | 课外活动 | 社团活动、讲座等课外活动记录 |
| 7 | 岗位追踪 | 求职目标岗位管理、技能需求对照、完成度追踪 |
| 8 | 数据分析 | 学期对比、同学对照、雷达图、GPA 趋势图 |
| 9 | AI 智能分析 | 基于全部数据的 AI 综合分析与建议 |
| 10 | 简历生成 | 基于个人数据自动生成 JSON/HTML 格式简历 |

**示例代码（CourseDao::create）：**

```cpp
bool CourseDao::create(Course& course) {
    QSqlQuery query(m_db);
    query.prepare(
        "INSERT INTO courses (name, code, credits, semester, category, score, "
        "grade_point, status, teacher, location, description, tags) "
        "VALUES (:name, :code, :credits, :semester, :category, :score, "
        ":grade_point, :status, :teacher, :location, :description, :tags)");
    query.bindValue(":name", course.name);
    query.bindValue(":code", course.code);
    query.bindValue(":credits", course.credits);
    query.bindValue(":score", course.score > 0 ? QVariant(course.score) : QVariant());
    query.bindValue(":grade_point", course.gradePoint > 0 ? QVariant(course.gradePoint) : QVariant());
    // ... 其余字段绑定
    if (!query.exec()) return false;
    course.id = query.lastInsertId().toInt();
    return true;
}
```

**AiService 核心流程：**

```
analyze()
  ├── 检查缓存（24h 有效期内直接返回）
  ├── 重置服务器状态 → resetAiServerCheck()
  ├── 探测本地 AI（GET /health，QNetworkAccessManager）
  │   ├── 成功 → analyzeWithAi() → callAiServer()
  │   └── 失败 → 探测远程 AI（GET /health）
  │       ├── 成功 → analyzeWithAi()
  │       └── 失败 → analyzeLocal() 本地模板分析
  └── 缓存结果
```

## 4. 关键技术实现

### 4.1 QTableWidget 表格管理

以 `CoursesPage` 为例，展示 Qt 表格的标准用法：

```cpp
QTableWidget* table = new QTableWidget(this);
table->setColumnCount(7);
table->setHorizontalHeaderLabels({"课程名称", "代码", "学分", "学期", "成绩", "绩点", "状态"});
table->setSelectionBehavior(QAbstractItemView::SelectRows);
table->setSelectionMode(QAbstractItemView::SingleSelection);
table->setEditTriggers(QAbstractItemView::NoEditTriggers);
table->horizontalHeader()->setStretchLastSection(true);
table->setSortingEnabled(true);
```

数据填充由 Service 层提供，通过 `CourseService::getAll()` 获取 `QList<Course>`，遍历后填入表格行。`setSortingEnabled(true)` 支持点击表头排序。

### 4.2 QSqlQuery 参数化查询

使用 `prepare()` + `bindValue()` 防止 SQL 注入：

```cpp
QSqlQuery query(m_db);
query.prepare("INSERT INTO courses (name, code, credits, ...) VALUES (:name, :code, :credits, ...)");
query.bindValue(":name", course.name);
query.bindValue(":code", course.code);
query.bindValue(":score", course.score > 0 ? QVariant(course.score) : QVariant());
query.exec();
```

注意 `score` 和 `grade_point` 字段使用 `QVariant()` 处理空值（映射到 SQL NULL）。

### 4.3 CrudPageController 策略模式

用 `std::function` 注册表替代传统的 switch-case：

```cpp
using CreateDialogFn = std::function<QJsonObject(QWidget*)>;
using RefreshFn = std::function<void()>;

// MainWindow 中的页面刷新注册表
QMap<int, std::function<void()>> m_pageRefreshers;
```

每种页面类型注册自己的创建对话框函数和刷新函数，新增页面类型只需注册，无需修改控制器代码。

### 4.4 DataRefreshCoordinator 星型拓扑

12 个页面不直接互相通信，而是通过协调器中转：

```cpp

    switch (domain) {
        case DataDomain::Courses:

            m_analysisPage->refresh();
            break;
        // ...
    }
}
```

### 4.6 AiService 双模式 AI + 上下文注入

支持本地 AI 和远程 AI 两种模式，自动降级：

```cpp
// 探测本地 AI（localhost:8001/health）
// 探测远程 AI（公网地址/health）
// 本地优先 → 远程降级 → 本地模板兜底
```

**AI 上下文注入（v2 新增）：**

AI 大模型模式下，系统会自动收集用户的真实学业数据并作为 `system message` 注入对话，使 AI 能基于具体数据给出针对性建议：

```cpp
// AiService::buildProjectContext() — 收集真实数据
QJsonObject AiService::buildProjectContext() {
    QJsonObject ctx;
    // 从 CourseService 收集 GPA、课程数、学分进度
    QJsonObject courseStats = CourseService::getStatistics();
    ctx["gpa"] = courseStats.value("gpa").toDouble();
    ctx["completedCredits"] = courseStats.value("completedCredits").toDouble();
    // 从 GoalService 收集目标进度
    // 从 ExperienceService 收集经历数量
    // 从 AchievementService 收集成就数量
    return ctx;
}

// chatWithAi() — 注入 system message
QJsonObject context = data.contains("context") ? data["context"].toObject() : buildProjectContext();
messages.append(QJsonObject{
    {"role", "system"},
    {"content", "以下是用户的真实学业数据，请基于这些数据回答问题：\nGPA: 3.52\n课程总数: 28\n..."}
});
```

**数据流：**

```
MainWindow::chatMessageSent
  → payload["context"] = AiService::buildProjectContext()
    → AiService::chat(payload)
      → chatWithAi(data)
        → buildProjectContext() 收集 GPA/课程/目标/经历/成就
        → 拼成 system message + user message
        → callAiServer("/v1/chat/completions", requestData)
          → Python ai_server.py → Qwen2.5 模型推理
```

**3. 追问上下文联动**

点击"追问"按钮后，选中文本自动注入 AI 对话上下文：

```
用户选中文字 → 点击"追问"
  → AiContextMediator::pushSelectionToPanel(text)
    → AiPanelWidget::addContextFromSelection(text)
      → AiConversationWidget::setSelectedContext(text)  // 显示上下文卡片
      → AiConversationWidget::focusChatInput()           // 聚焦输入框
  → 用户输入问题 → 发送
    → chatMessageSent("请结合以下选中文本回答...\n选中文本：xxx\n我的问题：yyy")
      → AiService::chat() + buildProjectContext()  // 同时注入真实数据
```

| Qt 类 | 用途 |
|-------|------|
| `QGraphicsDropShadowEffect` | 悬浮菜单阴影效果 |
| `QScreen` | 屏幕边界检测定位 |
| `QClipboard` | 复制功能 |
