# 运行说明

## 方式一：直接运行可执行文件

进入 `02_可执行文件/build_desktop`，双击：

```text
pdp_desktop.exe
```

如果需要后端服务，也可以运行：

```text
pdp_server.exe
```

## 方式二：从源码重新构建

进入 `01_源码` 后执行：

```powershell
set QT6_DIR=C:\Qt\6.11.0\mingw_64
.\build_desktop.bat
```
