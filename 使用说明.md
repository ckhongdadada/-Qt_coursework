# 独立HTML文件使用说明

## 文件信息

- **文件名**: `index_standalone.html`
- **大小**: 约 147 KB
- **包含内容**: 20个完整的演示幻灯片
- **依赖**: 无需任何外部文件或资源

## 特点

✅ **完全独立**: 所有样式、内容和脚本都包含在单个HTML文件中  
✅ **无外部依赖**: 不需要字体文件、CSS文件或JavaScript文件  
✅ **使用系统字体**: 采用微软雅黑、苹方等系统内置字体，无需下载  
✅ **即开即用**: 双击即可在任何浏览器中打开使用  
✅ **可移植**: 可以直接上传到任何平台或通过邮件发送  

## 使用方法

### 1. 本地使用
- 直接双击 `index_standalone.html` 文件
- 或在浏览器中拖放该文件

### 2. 在线分享
- 可以上传到任何静态网站托管服务（GitHub Pages、Netlify等）
- 可以作为邮件附件发送
- 可以上传到任何支持HTML文件的平台

### 3. 控制方式

#### 鼠标控制
- 点击底部的 **← 上一页** 和 **下一页 →** 按钮

#### 键盘快捷键
- `←` / `PageUp` - 上一页
- `→` / `PageDown` / `Space` - 下一页
- `Home` - 跳转到第一页
- `End` - 跳转到最后一页
- `1-9` - 快速跳转到第1-9页

## 如何生成这个文件

如果需要重新生成，运行以下命令：
```bash
python merge_slides.py
```

生成脚本会：
1. 读取 `slides/` 目录下的所有HTML文件
2. 提取每个幻灯片的核心内容
3. 移除外部字体依赖
4. 整合所有样式和内容
5. 添加翻页控制功能
6. 输出为单个HTML文件

## 故障排除

### 问题: 字体显示不正常
**解决方案**: 
- 确保使用的是现代浏览器
- Windows系统会自动使用微软雅黑
- macOS系统会使用苹方字体

### 问题: 页面显示不完整
**解决方案**:
- 确认浏览器窗口足够大（至少1280x720）
- 尝试全屏模式（按F11）

### 问题: 快捷键不工作
**解决方案**:
- 确保焦点在网页上（点击一下页面）
- 某些浏览器可能会拦截某些快捷键
