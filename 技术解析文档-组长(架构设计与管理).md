### 2.3 数据流

```
HTTP Request
    → QHttpServer (路由匹配)
    → XxxApi (请求解析、响应封装)
    → XxxService (业务逻辑)
    → XxxDao (数据库操作)
    → QSqlQuery (SQL执行)
    → SQLite (数据存储)
    → 返回结果沿原路返回
    → HTTP Response (JSON)
```

| 决策项 | 选择 | 备选方案 | 选择理由 |

### 5.4 HttpServer (server/HttpServer.h)

**架构角色：** 系统入口、路由注册中心、CORS中间件

**创建的核心对象：**

| 对象 | 类型 | 职责 |
|------|------|------|
| `m_server` | QHttpServer | HTTP请求处理引擎 |
| `m_tcpServer` | QTcpServer | TCP端口监听 |

**路由注册策略：**

| 策略 | 说明 |
|------|------|
| 精确路径优先 | `/api/courses/statistics` 先于 `/api/courses` 注册 |
| 参数路由 | 使用 `<arg>` 匹配动态ID |
| HTTP方法分离 | 同一路径不同方法分别注册 |
| CORS全局处理 | afterRequest中间件统一添加CORS头 |

**完整路由注册代码结构：**

```cpp
void registerRoutes() {
    m_server.afterRequest([](QHttpServerResponse&& resp) {
        resp.setHeader("Access-Control-Allow-Origin", "*");
        resp.setHeader("Access-Control-Allow-Methods", "GET, POST, PUT, DELETE, OPTIONS");
        resp.setHeader("Access-Control-Allow-Headers", "Content-Type, Authorization, X-Auth-Token");
        return std::move(resp);
    });

    m_server.route("/api/courses/statistics", Get, CourseApi::getStatistics);
    m_server.route("/api/courses", Get, CourseApi::getAll);
    m_server.route("/api/courses", Post, CourseApi::create);
    m_server.route("/api/courses/<arg>", Get, CourseApi::getById);
    m_server.route("/api/courses/<arg>", Put, CourseApi::update);
    m_server.route("/api/courses/<arg>", Delete, CourseApi::remove);
}
```

### 6.2 数据库初始化流程

```cpp
bool initializeDatabase() {
    QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
    db.setDatabaseName("pdp.db");
    if (!db.open()) return false;
    
    QFile schemaFile(":/resources/schema.sql");
    schemaFile.open(QIODevice::ReadOnly);
    QString schema = schemaFile.readAll();
    QStringList statements = schema.split(";", Qt::SkipEmptyParts);
    for (const auto& stmt : statements) {
        QSqlQuery query(db);
        query.exec(stmt.trimmed());
    }
    return true;
}
```

```cmake
cmake_minimum_required(VERSION 3.25)
project(pdp_server LANGUAGES CXX)

set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)
set(CMAKE_AUTOMOC ON)

find_package(Qt6 REQUIRED COMPONENTS Core Sql HttpServer Network)

add_executable(pdp_server
    src/main.cpp
    src/model/Course.cpp
    src/dao/DaoBase.cpp
)

target_link_libraries(pdp_server PRIVATE
    Qt6::Core Qt6::Sql Qt6::HttpServer Qt6::Network
)

target_include_directories(pdp_server PRIVATE ${PROJECT_SOURCE_DIR}/src)

file(COPY ${CMAKE_SOURCE_DIR}/resources/schema.sql
     DESTINATION ${CMAKE_BINARY_DIR}/resources)
```
