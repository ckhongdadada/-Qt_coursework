# 期末演示演讲稿与整体规划

## 一、项目概述

### 1.1 项目名称

**学业发展规划系统** (Personal Development Planning System)

### 1.2 项目简介

这是一个基于 Qt 6 构建的跨平台桌面应用，用于管理学业发展过程中的各类数据，包括课程、目标、角色、成果、经历、活动、岗位等，并提供数据分析、简历生成、AI 辅助等功能。

### 1.3 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| C++ | 17 | 核心语言 |
| Qt6 | 6.5+ | UI框架、HTTP服务器、数据库 |
| SQLite | 3.x | 嵌入式数据库 |
| CMake | 3.25+ | 构建系统 |


### 2.2 分层职责

| 层次 | 目录 | 职责 |
|------|------|------|
| 表现层 | src/client/ | UI展示、用户交互 |
| 控制层 | src/client/core/ | 跨组件协调、状态管理 |
| 服务层 | src/service/ | 业务逻辑封装 |
| 数据访问层 | src/dao/ | 数据库CRUD操作 |
| 模型层 | src/model/ | 数据模型定义 |
| 数据层 | SQLite/QSettings | 数据持久化 |

## 五、技术亮点

### 5.1 架构设计

1. **六层分层架构**：职责明确，易于维护
2. **单模式架构**：纯桌面应用，直接调用Service层
3. **协调器模式**：核心协调器管理跨组件交互
4. **Header-Only设计**：减少.cpp文件，简化构建

### 5.2 代码质量

1. **MainWindow重构**：从5000+行重构到766行
2. **统一响应格式**：JsonUtils确保API一致性
3. **CORS中间件**：前后端分离架构正常工作
4. **测试覆盖**：Model/DAO/Service层测试覆盖率95%+

### 5.3 性能优势

| 指标 | Flask版本 | C++版本 |
|------|----------|---------|
| 启动时间 | ~2s | ~0.1s |
| 内存占用 | ~50MB | ~10MB |
| 请求响应 | ~10ms | ~1ms |
