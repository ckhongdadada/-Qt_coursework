## 🚀 快速开始

### 1. 打开AI面板
- 位置：主窗口**右侧**
- 默认：展开状态（380px宽）
- 折叠：点击 **[收起]** 按钮

### 2. 获取建议（推荐）
```
1. 点击 [建议] 标签
2. 选择快捷按钮：
   • [综合] - 全面分析
   • [课程] - 课程建议
   • [经历] - 实践建议
   • [目标] - 目标策略
3. 查看结果
```

### 3. 对话交互
```
1. 点击 [对话] 标签
2. 在底部输入框输入问题
3. 点击 [发送] 或按 Enter
4. 查看AI回复
```

## 🎯 使用技巧

### ✅ 最佳实践

1. **先添加数据**
   - 确保已添加课程、目标、经历等数据
   - 数据越完整，建议越准确

2. **使用上下文**
   - 在列表中选中相关文本（5-2000字符）
   - AI会自动捕获并显示在状态栏下方
   - 点击快捷按钮时会结合上下文分析

3. **选择合适类型**
   - 课程规划 → [课程]
   - 职业发展 → [经历]
   - 目标管理 → [目标]
   - 全面分析 → [综合]

4. **及时回填**
   - 生成建议后，点击 [回填摘要] 应用到简历
   - 或点击 [转为目标] 创建新目标

### ❌ 常见错误

1. ❌ 没有数据就点击分析
   - ✅ 先添加课程、目标等数据

2. ❌ 选中无关文本作为上下文
   - ✅ 选中相关文本或点击 [清空上下文]

3. ❌ 期望对话模式有复杂推理
   - ✅ 使用快捷按钮获取标准化建议

4. ❌ 点击回填但没有生成建议
   - ✅ 先点击快捷按钮生成建议

## ⚠️ 常见问题

### Q1: 为什么上下文没有自动捕获？

**原因：**
- 文本太短（<5字符）或太长（>2000字符）
- 选中的不是文本内容
- 选中的是AI面板内的文本

**解决：**
- 选中5-2000字符的文本
- 在主内容区域选择
- 手动清空后重新选择

### Q2: 为什么点击快捷按钮没有反应？

**原因：**
- AI服务未启动
- 数据库中没有数据

**解决：**
- 点击 [刷新] 更新状态
- 确保已添加数据
- 查看状态是否显示"可用"

### Q3: 为什么回填摘要失败？

**原因：**
- 简历页面未初始化
- 没有生成AI建议

**解决：**
- 先切换到简历页面
- 生成AI建议后再回填
- 查看Toast通知的错误提示

### Q4: 为什么对话没有回复？

**原因：**
- 当前仅支持规则模式
- 输入的问题为空

**解决：**
- 使用快捷按钮（更可靠）
- 输入具体问题
- 等待后续版本支持LLM
