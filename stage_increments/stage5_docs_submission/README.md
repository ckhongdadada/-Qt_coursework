# 个人发展规划系统 - Stage 5

本仓库记录了一个 C++ 个人发展规划系统的分阶段开发过程。

Stage 5 是最终阶段，包含完整的文档、技术报告、演示材料和最终提交包。

## Stage 5 范围

- 技术架构文档 (`ARCHITECTURE.md`)
- 组件说明文档 (`COMPONENTS.md`)
- 开发指南 (`DEVELOPMENT_GUIDE.md`)
- AI 助手文档 (`AI_ASSISTANT_GUIDE.md`, `AI_ASSISTANT_QUICK_REF.md` 等)
- 变更日志 (`CHANGELOG.md`)
- 版本文件 (`VERSION`)
- 技术分工文档（各组员贡献说明）
- 软件体系结构与设计模式报告
- 期末演示演讲稿
- 后续改进与升级计划
